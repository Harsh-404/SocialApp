# SocialApp

## Project Description
SocialApp is a web application that allows users to connect and interact with each other in a virtual environment. The platform offers a range of features including user profiles, messaging, and content sharing, making it easy to build and maintain relationships online.

## Key Features
- User registration and authentication
- Personal profiles with customizable settings
- Real-time messaging
- Content sharing (text, images, videos)
- User groups and communities

## Live Link
You can access the live version of the application here: [Live Demo](https://socialapp-ui.onrender.com/)

## Technologies Used
- React
- Node.js
- Express
- MongoDB
- CSS

## Getting Started
To run the project locally:
1. Clone the repository
2. Install dependencies with `npm install`
3. Start the application with `npm start`
